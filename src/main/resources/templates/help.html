Cách làm bài thi

Tạo đề thi đầu tiên


  • Truy cập vào Giao diện tạo câu hỏi, nhập tên gói câu hỏi vào bấm "Thêm"

  • Tiếp tục chọn gói câu hỏi rồi chọn thêm câu hỏi.
    Tại giao diện thêm câu hỏi, có thể nhập thủ công các câu hỏi để thêm vào hoặc thêm bằng file excel( Hướng dẫn thêm bằng file excel )

  • Chuyển đến Giao diện quản lý đề
    Chọn 'Thêm' để thêm bộ đề
    Điền các thông tin cho bộ đề và nhấn "Xác nhận".

  • Quay lại Giao diện quản lý đề, chọn bộ đề vừa tạo và chọn thêm đề.

  • Sau khi đã tạo hoàn thiện 1 bộ đề, lúc này người dùng khác vẫn chưa thể nhìn thấy được bộ đề vừa tạo đó, nên chúng ta cần công khai bộ đề.
    Tại Giao diện quản lý đề, chọn bộ đề vừa tạo và nhấn "Công khai".

Upload câu hỏi bằng file excel


  • Trong Giao diện thêm câu hỏi, chọn Thêm bằng file excel sẽ dẫn đến giao diện thêm bằng file excel.

  • Nhấn tải xuống để tải file mẫu.
    Tiến hành chỉnh sửa lại file mẫu cho phù hợp , sau đó upload lên.
